## Content marketplace
This is a market place where users can get content

### Tools Used
- React
- React-Redux
- reactstrap - mostly
- framer-motion
- font-awesome
- node-sass
- victory
- react-albus
- react-tooltip
- eslint

### Setup Requirements
- node at least v10
- npm
- Your favourite browser

### Getting Started
- Clone this repo
- Install the dependencies, `npm install`
- start the development server, `npm start`
